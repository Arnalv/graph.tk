Graph.tk Help and Documentation

Getting Started

Graph.tk is easy and fast to use.

  1. Visit http://graph.tk
  2. Click the button to add a new function
  3. Type the equation using * for multiplication, / for division and ^ for exponentiation. To type special characters use "\name". For example, "\sqrt" yields √ and \pi produces a π.
